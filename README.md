## üìä Overview
This repository features the **Carnival Mega-Project Data Analysis**, an interactive Power BI dashboard (`Carnival Data Project - Nov25.pbix`) that consolidates comprehensive operational and economic data from the 2022 to 2025 Carnival editions (including projections for 2025).

The main objective is to provide a complete, multi-dimensional view of the event, covering economic impact, logistical structure, and cultural components. The analysis is structured into three core areas: **Executive Summary**, **Samba Schools**, and **Operational Structure/Equipment**.
